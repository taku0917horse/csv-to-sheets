CSV → Google スプレッドシート転記ツール

⚙️ 初回セットアップ手順（クリックで展開）
  1. Google Cloud Console でプロジェクトを作成し、Google Sheets API を有効化する
  2. 「認証情報」→「OAuth 2.0 クライアント ID」を作成（アプリの種類: ウェブアプリケーション）
  3. 「承認済みの JavaScript 生成元」に このHTMLを開いているオリジン（例: http://localhost:5500 や file://）を追加
  4. 作成した クライアント ID を下の設定欄に貼り付ける
  5. 転記先スプレッドシートのURLから スプレッドシート ID をコピーする
    例: https://docs.google.com/spreadsheets/d/[ここがID]/edit
設定
Google Cloud Console の「認証情報」から取得
